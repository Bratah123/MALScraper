# MALScraper
A Script that scrapes the name and img from MyAnimeList

# Run it
While I already provided the list of the first 5000 characters, if you wish to use it yourself
Run: `python scraper.py`

## JSON Format
```json
[
  {
     name : ["Itadori", "Yuuji"],
     img : "link.com",
     anime: "Jujutsu Kaisen"
  }
]
```
